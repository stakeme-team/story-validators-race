## Story Validator Grafana Dashboard




### Grafana Dashboard is a visualization and analytics tool used to monitor and analyze data from your hardware metrics and Validator health metrics. 




My dashboard showing:
```
Story Network Info
	- Network name
	- Active Validators
	- TotalValidators
	- Alias
	- Api Height
Story rpc node info
	- Sync status
	- Geth sync status
	- peers number
	- Geth peers number
	- rpc node height
Validator Info
	- Moniker
	- valoper
	- Bonded tokens
	- jailed
	- tombstaned
	- delegators
ServerInfo
	- Total CPU
	- Cpu used
	- Totel RAM
	- RAM Used
	- Uptime
	- Disk space used
	- Network traffic
	- Disk IOps

```
